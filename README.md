# Edvin HUGO page

### Add a new site
```hugo new site edvin_hugo```

### Add the goa theme
```git submodule add https://github.com/shenoybr/hugo-goa.git themes/goa```

### Start hugo
```hugo server -w -v```

### Create a post, this contains some metadata
```hugo new content/blog/my-first-post.md```
